# Registers in iAPX88

## General Registers

- AX -- Accumulator Register
- BX -- Base Register
- CX -- Counter Register
- DX -- Destination Register

## Other Registers

| Segment Registers   | Pointer Registers         | Index Registers         |
| ------------------- | ------------------------- | ----------------------- |
| CS -- Code Segment  | IP -- Instruction Pointer |
| DS -- Data Segment  | SI -- Source Index        | DI -- Destination Index |
| SS -- Stack Segment | SP -- Stack Pointer       | BP -- Base Pointer      |
| ES -- Extra Segment |

## Flags

> **[- - - - O D I T S Z - A - P - C]**

- C -- Carry Flag
- P -- Parity Flag
- A -- Auxiliary Flag
- Z -- Zero Flag
- S -- Sign Flag
- T -- Trap Flag
- I -- Interrupt Flag
- D -- Direction Flag
- O -- Overflow Flag
